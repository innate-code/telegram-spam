Спамим в Telegram помощью termux

Итак, первым делом нам необходимо получить наши данные для авторизации в скрипте. Первым делом заходим на сайт (https://my.telegram.org/auth), вводим свой номер телефона и вводим код подтверждения (делайте это на свой страх и риск с левого аккаунта) 

Теперь мы переходим в раздел API development tools. Появится окно - "Создать новое приложение", указываем название приложения и краткое имя, это будет вполне достаточно, и нажимаем создать приложение. У нас появится окно в котором будут указаны: appip, apihash. Копируем их и сохраняем куда-то, они нам еще понадобятся.

Далее мы переходим уже в Termux

Первым делом мы обновляем наши пакеты:
pkg update
pkg upgrade

Далее устанавливаем необходимые инструменты: 
pkg install git
pkg install python
pip install -U telethon --user

Скачиваем наш скрипт:
git clone https://github.com/SergoProxy/TG-spam

На этом наша подготовка закончена теперь мы запускаем скрипт. 

1. Переходим в домашнюю директорию (команда cd)

2. Далее смотрим список файлов и папок командой ls.

3. Командой переходим в папку скачанного скрипта TG-spam:
cd TG-spam

4. И запускаем наш скрипт:
python spamtg.py

5. Теперь мы вводим всё то что нас просит скрипт:
Введите хэш аккаунта - это ваш apphash - вводим его
Введите ip приложения - это ваш appip
Введите id пользователя - нам необходимо узнать id пользователя, именно ID, не путайте с именем пользователя, сделать это можно с помощью бота телеграмм @getmyid_bot. Для этого мы пересылаем любое сообщение жертвы этому боту и получаем id жертвы.

6. И при первой авторизации скрипт нас попросит ввести номер телефона и код подтверждения. 

Всем сочного спама, друзья
